# Snoraa - Hotel Booking Application 

A premium, modern hotel booking platform built with **React** and **Vite**. Snoraa offers a seamless experience for users to discover, filter, and book hotels with a polished UI and intuitive user flow.

![Snoraa Demo](public/snoraa_logo_icon.png)

##  Features

-  Interactive Listings**: Grid and List view toggles for flexible browsing.
-  Advanced Search & Filtering**: Filter by price, amenities, rating, and property type.
-  Wishlist**: Save your favorite properties for later.
-  Dark/Light Mode**: Fully themeable UI with a toggle switch.
-  Skeleton Loading**: Smooth loading states for a better user experience.
-  PDF Ticket Download**: Generate and download booking confirmations instantly.
-  Fully Responsive**: Optimized for desktop, tablet, and mobile devices.

##  Tech Stack

- Frontend: React.js, Vite
- Styling: Vanilla CSS (Variables & Responsive Design), Google Fonts ('Outfit')
- Icons: Lucide React
- Notifications: React Hot Toast
- State Management: Context API (Auth, Wishlist, Theme)

##  Getting Started

1.  **Clone the repository**
    ```bash
    git clone https://github.com/navyaaggarwal29/snoraa-hotel-booking.git
    ```
2.  **Install dependencies**
    ```bash
    cd snoraa-hotel-booking
    npm install
    ```
3.  **Run the development server**
    ```bash
    npm run dev
    ```
4.  Open `http://localhost:5173` to view it in the browser.
